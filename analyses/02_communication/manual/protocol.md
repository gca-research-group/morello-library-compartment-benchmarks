# Manual protocol

1. Execute the communication benchmark and collect raw CSV files.
2. Restrict the analysis to the message sizes reported in the paper.
3. Apply the outlier-removal rule described in the manuscript.
4. Compute mean latency, standard deviation, coefficient of variation, throughput, and latency overhead.
5. Archive the resulting processed summaries as CSV/XLSX files.

The exact published processed summaries are preserved in `../published/`.
