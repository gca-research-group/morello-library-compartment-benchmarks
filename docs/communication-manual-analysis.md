# Communication Manual Analysis

The communication benchmark described in Section 4.2 is preserved in two layers:

- raw benchmark artefacts under `experiments/02_communication/`
- preserved manual-analysis outputs under `analyses/02_communication/`

This repository **does not claim** that the exact published communication table can be regenerated automatically from the raw CSV files using an authoritative script. Instead, it preserves the processed CSV/XLSX artefacts used in the communication analysis and documents that this part of the workflow was manual.

Consequently:

- the benchmark itself is **repeatable**,
- the processed summary is **archived and inspectable**,
- but the communication analysis is only **partially reproducible** end-to-end.
