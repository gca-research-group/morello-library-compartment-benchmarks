# Dataset Provenance

This repository distinguishes three levels of artefacts:

1. **Raw experiment outputs** — files produced directly by the benchmark programs (`experiments/*/raw/`).
2. **Derived analytical artefacts** — files produced by analysis scripts or preserved manual summaries (`analyses/*/`).
3. **Paper-facing copies** — convenience copies mapped directly to the manuscript tables/figures (`paper/tables/`, `paper/figures/`).

For integrity verification, see `MANIFEST.sha256`.
